Class: Button

Button.Button(x, y, w, h, ropt, text, styles, cb)

new Button(x, y, w, h, ropt, text, styles, cb)

Creates an interactive Button
Parameters:
Name Type Attributes Description
x Number X pos
y Number Y pos
w Number Width of button
h Number Height of button
r Number <optional>
 Radius of button, optional, required if shape is roundrect
text String Text to display in the button
styles Object Object containing all the styles for the button and the text
Properties
Name Type Attributes Description
shape String Shape of button, rect, roundrect, or image
color String Hex color(with "#"), rgb, or rgba
imgPath String <optional>
 Img path of button, only required if shape is image
stroke Object <optional>
 Object containing stroke styles for button
Properties
Name Type Attributes Description
style String <optional>
 Color of stroke
width Number <optional>
 Width of stroke
text Object Object containing all text styles
Properties
Name Type Description
font String Font of text in CSS style
color String Hex Color of text with "#"
method String Text render method, valid: fill, stroke, fillstroke
cb function onclick callback
