Particle.Particle(graphic, shapeopt, wopt, hopt, ropt)

Creates an instance of a Particle
Parameters:
Name Type Attributes Description
graphic String Hex color without "#", link, or file path.
shape String <optional>
 Shape of particle. Circle, Rect, or RoundRect. If graphic is a link/file shape is ignored. Optional.
w String <optional>
 Width of rect or round rect. Required for rect and round rect. Ignored if graphic is link/file.
h String <optional>
 Height of rect or round rect. Required for rect and round rect. Ignored if graphic is link/file.
r String <optional>
 Radius of round rect or circle. Required for circle and round rect. Ignored if graphic is link/file.
